# 交付物目录说明

本目录用于集中保存数据库大作业“寻根溯源族谱管理系统”的最终提交材料。

## 目录结构

- `实现方案与计划.md`
  项目需求分析、系统设计思路、分阶段实施计划。
- `report/`
  实验报告、性能分析、演示说明等文档。
- `sql/`
  建表、查询、索引、触发器、序列同步等 SQL 脚本。
- `scripts/`
  数据生成脚本与辅助工具源码。
- `data/`
  生成后的 CSV 数据文件与数据说明。
- `export/`
  数据库导入导出示例、备份说明、分支导出 SQL 示例。
- `screenshots/`
  界面截图清单、SQL 结果截图说明和答辩时需要准备的截图项。

## 当前内容说明

本次已补齐以下可直接提交的材料：

- 完整数据库脚本
- 样例与大规模数据生成脚本
- 数据导入/导出说明
- 实验报告 Markdown 版
- 性能测试记录模板与分析说明
- 截图清单与答辩演示建议

## 提交前建议检查

1. 如需现场演示 PostgreSQL，请先按 `sql/` 中脚本顺序初始化数据库。
2. 如需展示 10 万级数据，请先运行 `scripts/generate_data.py` 生成 CSV。
3. 如需提交截图，请按照 `screenshots/截图清单.md` 补充真实截图文件。
4. 如需提交数据库备份，请将导出的 `.sql` 或 `.backup` 文件放入 `export/`。
